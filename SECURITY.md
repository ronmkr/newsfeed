# Security Policy

## Supported Versions

The following versions of Unbiased India News are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of Unbiased India News seriously. If you believe you have found a security vulnerability, please do not report it via a public issue.

Instead, please report security vulnerabilities by emailing **raunak.jyotishi@gmail.com**.

Please include the following in your report:
- Type of issue (e.g., credential leak, logic flaw)
- The location of the affected code
- A description of the vulnerability
- Steps to reproduce

We will acknowledge receipt of your report within 48 hours and provide a timeline for a resolution.
